FluidVids.js

Introducing the Script

A lightweight, easy-to-use script for fluid width video embeds.

Todd Motto got tired of plug-ins and decided to create a much easier way to embed video in your pages, while keeping them responsive. Scripts are by far more effective. He also shows a step-by-step of creating the script

How Do I Use It?

Just drop the JavaScript file into your page (this needs to be placed before the closing tag or inside a DOM ready function wrapper) and let it work its magic. No config required. Minified version also included in the download. <script src="path/to/fluidvids.js"></script> That's it! You don't need to do anything else in your HTML or CSS
